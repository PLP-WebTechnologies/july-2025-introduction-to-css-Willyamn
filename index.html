Box Model Basics

The CSS Box Model is the foundation of web layout. Each element is a rectangular box, consisting of:

  • Content – the text or images inside the element
  • Padding – the space between content and border
  • Border – wraps around the padding
  • Margin – the outermost layer, spacing between elements
